About Me

Let me introduce myself.

I got into programming while teaching a computer science class for 4th grade students. After that, I started using no-code tools to build a class game app for my classroom, and managing it with Google Sheets inspired me to learn Python. That eventually led me to web design, where I began working with HTML and CSS.

Since then, I’ve earned my CompTIA A+ certification and am working on my Network+. I enjoy experimenting in my homelab, exploring networking and other tech projects. My goal is to create websites that are functional, engaging, and fun to use while I progress on my tech journey.
