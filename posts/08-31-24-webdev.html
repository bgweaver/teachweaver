Bubbles & Fur Website

Posted by Ben Weaver on August 31, 2024

As a web developer, each project offers new challenges and opportunities to grow. My most recent project, a website for Bubbles & Fur, was no exception. This was my first time creating a site for someone else, and I’m proud of the progress I’ve made. I wanted to take a moment to reflect on what went well and where I see room for improvement.

*Note: This version has undergone several updates and is no longer live. You can see the original archived on GitHub. You can also see the latest version live at Bubbles & Fur or on GitHub.

What I’m Proud Of:

  • First Published Site for a Client: This was my first official project for someone else, and seeing it live was incredibly rewarding. Delivering a site that meets a client’s needs feels like a milestone.
  • Utilized Netlify for Deployment: This was my first time using Netlify, which made the deployment process smooth and efficient.
  • Form Integration: I implemented Netlify Forms with honeypot and CAPTCHA features to reduce spam, a critical improvement from past experiences.
  • Custom Email Setup: Using ImprovMX, I set up a custom email address, enhancing the client’s brand credibility.
  • Animation with AOS: Subtle animations added a dynamic element, making the first impression more engaging.
  • Meeting the Client Brief: I adapted quickly to client requests, ensuring the final product aligned with her vision.
  • About Page Creation: I crafted an “About the Business” page using the client’s interview responses, adding a personal touch.
  • Expanded Use of Bootstrap: I ensured responsiveness across devices and seamless navigation bar behavior.
  • Meta Tags for Enhanced Sharing: Implementing meta tags improved how the site appears on social media and messaging apps.
  • Site Indexing and Google Business Entry: Indexing the site improved its visibility and online presence.

Wishes and Next Steps:

  • Integrating Payment: The next step is adding a payment system like Clover or PayPal, which will be a learning opportunity.
  • Improving Bootstrap Skills: I want to enhance my layouts and solve centering issues more effectively.
  • Enhancing Visual Appeal: Some pages lack visual interest, so I’ll focus on improving design principles.
  • Expanding My Client Base: My goal is to publish a site for a non-relative client, expanding my portfolio.
  • Creating a Portfolio Site: I aim to build a custom site showcasing my work and skills.

This project was a significant step forward in my web development journey. I’m excited about the progress I’ve made and motivated by the challenges ahead. Each project is a learning experience, and I can’t wait to see where the next one takes me.
